Graphical User Interface (GUI)
Command Line Interface (CLI)

- Make Directory
mkdir [directory_name]

- Delete Directory
rmdir [directory_name]

- Print/Present Working Directory
pwd

- List all files and folder
1s

- Change Directory
cd [directory_path]

- Go back to the parent directory
cd ..

- Create file
touch [filename]

- Remove file
rm [filename]

- Clear CLI
clear
